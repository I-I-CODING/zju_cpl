# 字符串
- eg
`char word[] = {'H','E','L','L','O','!','};` 是字符数组，不是字符串
`char word[] = {'H','E','L','L','O','!','\0'};`
是字符数组，也是字符串

- 以0结尾的一串字符
- 0 和 '\0'一样，但和'0'不同
- 0标志字符串的结束，但其不是字符串的一部分
- 计算字符串长度时不包括0
- 字符串以数组的形式存在，以数组或指针的形式访问
- `string.h`放在头文件里，用于处理字符串函数

# 字符串变量
几种表达方式：
- `char *str = "hello";`
- `char worc[] = "hello";`
- `char line[10] = "hello";`

# 字符串常量
- `"hello"`
- `"hello"`会被编译器变成一个字符数组放在某处，这个数组的长度是6，结尾还有表示结束的0
- 两个相邻的字符串常量会被自动连接起来

# 其他
- c语言的字符串是以字符数组的形态存在的
- 不能用运算符对字符串做运算
- 通过数组的方式可以遍历字符串

# 字符串常量
`char *s = "hello world";`
- s是一个指针，初始化为指向一个字符串常量
- 这个字符串只读不写
- eg:
```c
char *s = "hello world";
s[0] = 'B';
printf("s[0] = %c\n", s[0]); 
// 编译出错
```
- 若要修改字符串，应该用数组：
`char s[] = "hello world";`

# 用指针和数组的情况
```c
char *str = "hello"; // 指针
char word[] = "hello"; // 数组
```
- 数组：
1. 字符串在这里
2. 作为本地变量空间自动被回收
- 指针
1. 不知道字符串在哪
2. 处理参数
3. 动态分配空间
