# 指针
- 一些表达式：
```c
int* p = &i;
int* p,q;
int *p,q;
```

第二行和第三行内容完全一样，一个指针变量和一个整型变量

- 指针变量
    - 普通变量的值是实际的值
    - 指针变量的值是具有实际值的变量的地址

- 作为参数的指针
`voidf(int *p)`
`int i = 0;`
`f(&i)`, 把变量的地址传给f函数

- 访问地址上的变量
    - *是一个单目运算符，用来访问指针的值所表示的地址上的变量
    - 可以做左值也可以做右值，如：`int k = *p;` `*p = k + 1;`

- 修改地址上的变量
    - 可以把`*p`看作整数
    - 修改`*p`的值，函数外的值也会被修改
```c
int main(void)
{
    int i = 0;
    f(&i);
}

void f(int *p)
{
    *p = 1;
}
```

此时输出i的结果， `i = 1`
