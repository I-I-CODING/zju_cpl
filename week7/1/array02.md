## 数组的集成初始化
```c
int a[]{1, 2, 3};
```
数组大小自动识别为3
```c
int a[3]{1};
output:
1 0 0
```

即空位补0

## 集成初始化时的定位
```c
int a[10]{[0] = 2, [2] = 3, 6};
output:
2 0 3 6 0 0 0 0 0 0
```

- 用[n]在初始化数据中给出定位
- 没有定位的数据接在前面的的位置后面
- 其他位置的值补0
## 数组的大小
- sizeof给出整个数组所占的内容的大小，单位是字节
- `sizeof(a[0])`给出数组中单个元素的大小
- 数组的大小：
```c
sizeof(a)/sizeof(a[0])
```

## 数组的赋值
- 错误的写法：`int b[] = a;`
- 要把一个数组的所有元素交给另一个数组，必须遍历
```c
for (i = 0; i < length; i++)
{
    b[i] = a[i];
}
```

## 学过的几种遍历数组的情况
- 数组赋值
```c
for (i = 0; i < length; i++)
{
    b[i] = a[i];
}
```

- 数组初始化为0
```c
for ( i = 0; i < number; i++)
{
    count[i] = 0;
}
```

- 查找数字
```c
for ( i = 0; i < length; i++)
{
    if (a[i] == key)
    {
        ret = 1;
        break;
    }
}
```

- 输出较大的数字
```c
for ( i = 0; i <cnt; i++)
{
    if (number[i] > averge)
    {
        printf("%d", number[i]);
    }
}
```

- 输出对应位数的数字
```c
for ( i = 0; i < number; i++)
{
    printf("%d:%d\n", i, count[i]);
}
```

- 注意：离开循环后， 不能继续用i的值做下标，因为此时`a[i]`已经越界
